## âœ¨ Fitur Unggulan

1. **Desain Modern & Responsif**
   - Tampilan Full-width dengan hero slider.
   - Responsif di semua perangkat (Mobile, Tablet, Desktop).
   - Animasi smooth saat scroll dan interaksi.

   - Sticky navbar dengan efek glassmorphism.

   - Filter kategori (Pembelajaran, Ekskul, Fasilitas).

   - Validasi form real-time.
   - Integrasi WhatsApp untuk pendaftaran siswa baru.
   - Tampilan form yang rapi dan user-friendly.
